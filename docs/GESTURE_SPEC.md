# AI小组件 · 手势交互规范

> v2.0 | 2026-03-23 | 参考：Apple HIG、Material Design 3、Android Automotive Design Guidelines

---

## 一、手势语义表

| 手势 | 语义 | 触发条件 | 适用场景 |
|------|------|----------|----------|
| **点击 (Tap)** | 主操作 / 进入详情 | 单次触碰 ≤300ms | 按钮、卡片内容区域、列表项 |
| **左右滑 (Swipe H)** | 切换同层内容 | 水平滑动 ≥60px | 车端卡片切换、列表项滑出操作按钮 |
| **上下滑 (Swipe V)** | 滚动内容 | 垂直滑动 | 新闻流、日程列表、闹钟列表 |
| **长按 (Long Press)** | 进入编辑态 / 危险操作确认 | 按住 ≥600ms 无移动 | 车端→编辑态、日程/闹钟删除 |
| **长按拖动 (Drag)** | 排序 | 长按后拖动 | 车端编辑态卡片排序 |
| **左滑（列表项）** | 显示操作按钮 | 列表项上水平左滑 ≥80px | 闹钟/日程→滑出删除按钮 |

**禁止手势**：
- 双指缩放（车载触屏不适用）
- 双击 / 三击（不符合车载单次操作原则）
- Hover 状态（触屏无鼠标）

## 二、HMI 安全约束

| 约束 | 标准值 | 说明 |
|------|--------|------|
| 最小触控目标 | **76×76dp** | 所有按钮、FAB、切换控件 |
| 最小正文字号 | **24dp** | body text、列表项文字 |
| 最大阅读文字量 | **≤20 字** | 驾驶中禁止需长时间阅读的操作 |
| 操作反馈 | 视觉 + 触觉 | 点击高亮 + haptic feedback |
| 主操作 | 单次点击完成 | 不允许要求连续多步才能完成主操作 |

## 三、各场景手势映射

### 车端模拟器 (car-simulator)

| 状态 | 手势 | 行为 |
|------|------|------|
| 显示态 | 左右滑 | 切换卡片 |
| 显示态 | 点击 | 进入卡片内交互 |
| 显示态 | 长按 600ms | → 编辑态 |
| 编辑态 | 点击卡片 | → 显示态 |
| 编辑态 | 左右滑 | 缩略图级浏览 |
| 编辑态 | 点击 ✕ | 删除确认弹窗 |
| 编辑态 | 长按拖动 | 卡片排序 |
| 编辑态 | 点击"完成" | → 显示态 |

### 天气卡片

| 手势 | 行为 |
|------|------|
| 点击城市名 | 弹出城市搜索 overlay |
| 上下滑 | 滚动 3 日预报区域 |
| 点击搜索结果项 | 选择城市、关闭 overlay、刷新天气 |

### 新闻卡片

| 手势 | 行为 |
|------|------|
| 上下滑 | 滚动新闻流 |
| 点击新闻项 | 展开全屏详情 overlay |
| 详情内左上角返回 | 关闭详情 overlay |
| 顶部下拉 | 触发刷新（live 模式） |

### 闹钟卡片

| 手势 | 行为 |
|------|------|
| 点击模式切换 | 列表 ↔ 时钟模式 |
| 列表项左滑 ≥80px | 显示红色"删除"按钮 |
| 点击"删除"按钮 | 删除该闹钟 |
| 点击 toggle 开关 | 启用/禁用该闹钟 |
| 点击 + 按钮 | 弹出新建闹钟 overlay |

### 日程卡片

| 手势 | 行为 |
|------|------|
| 上下滑 | 滚动日程列表 |
| 点击 FAB (+) | 弹出新建日程 overlay |
| 点击 ✅ 按钮 | 标记日程完成 |
| 长按日程项 | 弹出删除确认 |

### 音乐卡片

| 手势 | 行为 |
|------|------|
| 点击 ▶/⏸ | 播放/暂停 |
| 点击 ⏮/⏭ | 上一曲/下一曲 |
| 点击进度条 | 跳转到对应位置 |

### 纪念日卡片（恋爱/宝宝/放假/暖橙）

| 手势 | 行为 |
|------|------|
| 点击卡片任意区域 | 触发彩蛋粒子（从点击位置辐射） |

## 四、手势冲突处理

| 冲突场景 | 解决方案 |
|----------|----------|
| 闹钟列表项左滑 vs 车端卡片左右滑 | iframe 内 touch 事件不冒泡到父级，两者独立 |
| 新闻上下滑 vs 页面滚动 | 新闻容器 `overflow-y: auto` + `touch-action: pan-y`，阻止外层滚动 |
| 长按进入编辑态 vs 长按删除日程 | 车端长按作用于 car-screen（iframe 外），日程长按作用于 iframe 内 |
| 编辑态下的卡片点击 vs 卡片内点击 | 编辑态下卡片 80% 缩小，点击事件由编辑态拦截，不传递到 iframe |
